# vue3-element-plus-component
`element-plus` 组件封装，基于 vue3 + vue-router4 + vite + element-plus + ts 开发，如有什么建议和问题，欢迎提 `issues` 和 `pr` 。

## 功能
✅ 图标选择
✅ 省市区选择
✅ 趋势标记
✅ 时间选择
✅ 消息通知
✅ 城市选择
✅ 进度条
✅ 日历
✅ 表单
✅ 弹出框表单
✅ 表格
✅ 导航菜单
✅ Layout
